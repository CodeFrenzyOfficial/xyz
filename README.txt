To run the project, you have to follow thw below instructions 

you must have installed node and npm in your system 
If you dont have these

go to this site and download and install node latest version 
https://nodejs.org/en/download

After installing node
you have to install NPM, Run the following command in your Windows terminal to install npm

npm install -g npm 


After your node and npm has been installed 
you have to open your project in editor 
and run following command to install dependencies

npm install


After installing dependencies in your project, run following command to run your project

npm run start

and you are good to go 
your project will be live on
http://localhost:3000/
